**タスク:** 数理論理学(戸次大介)などを参考にまとめる — SKエイリアスの追加と体系情報の整理

### 周辺情報

- 戸次本では SK (S公理+K公理+MP) が独立した体系として扱われている
- 現在の実装では Minimal Logic (HM) = SK だが、SKの名称は出ていない
- HM ≈ SK, HJ = HM+EFQ, HK = HM+DNE = HM+EFQ+CM* = HM+CON3 = HM+CON4 等の等価性が整理可能
- 自然演繹(NM, NJ, NK)やシーケント計算(LM, LJ, LK)は現アーキテクチャでは大きな拡張が必要
- 今回はヒルベルト流体系のプリセットにSKエイリアス追加と、体系説明の充実を行う
