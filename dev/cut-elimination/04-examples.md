# カット除去の具体例

本ドキュメントでは、カット除去アルゴリズム（[03-algorithm.md](./03-algorithm.md)）の動作を具体的な証明で追跡する。各ステップでどの変換ケース（Case 1〜4）が適用されたかを明記し、アルゴリズムの理解を確認する。

> **前提知識:**
>
> - シーケント計算（LK）の推論規則: [01-sequent-calculus.md](./01-sequent-calculus.md)
> - カット除去定理の概要: [02-cut-elimination-theorem.md](./02-cut-elimination-theorem.md)
> - カット除去アルゴリズムの疑似コード: [03-algorithm.md](./03-algorithm.md)

> **推論規則のリマインダー（本ドキュメントで使用するもの）:**
>
> ```
> ─────── (Ax)          Γ ⊢ Δ                   Γ ⊢ Δ
> A ⊢ A              ────────── (WL)          ────────── (WR)
>                     A, Γ ⊢ Δ               Γ ⊢ Δ, A
>
>   Γ, B ⊢ Δ                  Γ ⊢ Δ, B     C, Σ ⊢ Π
> ──────────── (→R)         ────────────────────────────── (→L)
> Γ ⊢ Δ, B→C                  B→C, Γ, Σ ⊢ Δ, Π
>
> Γ ⊢ Δ, B    Γ ⊢ Δ, C          B, Γ ⊢ Δ
> ──────────────────────── (∧R)  ──────────── (∧L₁)
>     Γ ⊢ Δ, B∧C               B∧C, Γ ⊢ Δ
>
>   A[t/x], Γ ⊢ Δ               Γ ⊢ Δ, A[y/x]
> ──────────────────── (∀L)    ──────────────────── (∀R, y は Γ,Δ に自由でない)
>  ∀x.A, Γ ⊢ Δ                 Γ ⊢ Δ, ∀x.A
>
> Γ ⊢ Δ, A    A, Σ ⊢ Π
> ─────────────────────── (Cut)
>      Γ, Σ ⊢ Δ, Π
> ```

## 1. 例1: 公理とのカット（Case 1 → Case 2a → Case 1a）

### 1.1 証明対象

```
A ⊢ B → A
```

### 1.2 カットを含む証明

原子式 `A` をカット式として、不必要にカットを経由する:

```
                        A ⊢ A (Ax)
                     ────────────── (WL)
  A ⊢ A (Ax)         B, A ⊢ A
────────────── (WR) ────────────── (→R)
A ⊢ A, B → A        A ⊢ B → A
──────────────────────────────────── (Cut: A)
            A ⊢ B → A
```

- **カット式:** `A`（原子式、複雑度 0）
- **カット階数:** 0

### 1.3 カット除去の実行

#### ステップ1: カットの分析

- **カット式:** `A`（原子式）
- **左前提の最終規則:** `WR`（後件に `B → A` を追加）
- カット式 `A` は WR で追加された式ではなく、WR の前提 `A ⊢ A` の後件にある元の式
- **適用ケース:** **Case 2a（非主式、左を上に持ち上げ）**

WR はカット式 `A` に触れていないため、WR をカットの外側に持ち上げる。

#### ステップ2: Case 2a の適用

WR の前提 `A ⊢ A` を新たな左前提としてカットを構成し、WR を外側に移す:

```
A ⊢ A (Ax)     A ⊢ B → A
──────────────────────────── (Cut: A)
         A ⊢ B → A
     ─────────────────── (WR: B → A を追加)
     A ⊢ B → A, B → A
```

ここで新しいカット `A ⊢ A` と `A ⊢ B → A` のカットが残っている。

#### ステップ3: Case 1a の適用

新しいカットの左前提は公理 `A ⊢ A`:

```
A ⊢ A (Ax)     A ⊢ B → A
──────────────────────────── (Cut: A)
         A ⊢ B → A
```

- **左前提:** `A ⊢ A` — 公理
- **適用ケース:** **Case 1a（左が公理）**

Case 1a: 左前提が公理の場合、右前提の証明がそのまま結果になる。

結果: `A ⊢ B → A`（右前提の証明がそのまま）

#### 最終結果: カットなし証明

```
   A ⊢ A (Ax)
────────────── (WL)
  B, A ⊢ A
────────────── (→R)
 A ⊢ B → A
```

カットを除去した結果、元の右前提の証明そのものが得られた。

**まとめ:**

| ステップ | 適用ケース                          | カット式 | 複雑度 |
| -------- | ----------------------------------- | -------- | ------ |
| 1        | Case 2a（非主式、左を上に持ち上げ） | A        | 0      |
| 2        | Case 1a（公理とのカット）           | A        | 0      |

## 2. 例2: 含意の主式カット（Case 3.2）

### 2.1 証明対象

```
A ⊢ A
```

### 2.2 カットを含む証明

`A → A` をカット式として、不必要にカットを経由する:

```
                                     A ⊢ A (Ax)
                                   ────────────── (WL)
  A ⊢ A (Ax)              A ⊢ A (Ax)    A, A ⊢ A
────────────── (→R)      ──────────────────────────── (→L: A → A)
 ⊢ A → A                      A → A, A ⊢ A
──────────────────────────────────────────────── (Cut: A → A)
                    A ⊢ A
```

**→L の確認:**

`→L` で `B → C = A → A`（B = A, C = A）を分解:

- 左前提: `A ⊢ A`（Γ = {A}, Δ = ∅。B = A が後件にある）
- 右前提: `A, A ⊢ A`（C = A が前件、Σ = {A}, Π = ∅）
- 結論: `A → A, A ⊢ A`

- **カット式:** `A → A`（含意、複雑度 1）

### 2.3 カット除去の実行

#### ステップ1: カットの分析

- **カット式:** `A → A`（含意、複雑度 1）
- **左前提の最終規則:** `→R` — カット式 `A → A` は `→R` で導入 → **主式**
- **右前提の最終規則:** `→L` — カット式 `A → A` は `→L` で分解 → **主式**
- 両方で主式 → **Case 3.2（含意の主式カット）**

#### ステップ2: Case 3.2 の適用

カット式 `B → C`（B = A, C = A）の分解:

```
左の →R の前提:   A ⊢ A        (Γ = ∅, B = A, Δ = ∅, C = A)
右の →L の左前提: A ⊢ A        (Σ = {A}, Π = ∅, B が後件にある)
右の →L の右前提: A, A ⊢ A     (C = A が前件, Σ' = {A}, Π' = ∅)
```

Case 3.2 の変換: 2つの小さいカットに分解

```
第1カット Cut(C = A):
  左:  →R の前提      = A ⊢ A
  右:  →L の右前提    = A, A ⊢ A

第2カット Cut(B = A):
  左:  →L の左前提    = A ⊢ A
  右:  第1カットの結果
```

**変換後の証明:**

```
                    A ⊢ A (Ax)    A, A ⊢ A (WL of Ax)
                  ──────────────────────────────────── (Cut: A)
A ⊢ A (Ax)                    A ⊢ A
──────────────────────────────────────── (Cut: A)
               A ⊢ A
```

両方のカット式は `A`（原子式、複雑度 0）。元のカット式 `A → A` は複雑度 1 だったので、カット式の複雑度が **厳密に減少**。

#### ステップ3: 第1カットの除去（Case 1a）

```
A ⊢ A (Ax)    A, A ⊢ A
──────────────────────── (Cut: A)
         A ⊢ A
```

- **左前提:** `A ⊢ A` — 公理
- **適用ケース:** **Case 1a（左が公理）**

Case 1a: 右前提の前件からカット式 `A` の1つの出現を取り除いた証明が結果。
`A, A ⊢ A` は `A ⊢ A` の WL なので、取り除くと `A ⊢ A`。

結果: `A ⊢ A`

#### ステップ4: 第2カットの除去（Case 1a）

```
A ⊢ A (Ax)    A ⊢ A
──────────────────────── (Cut: A)
         A ⊢ A
```

- **左前提:** `A ⊢ A` — 公理
- **適用ケース:** **Case 1a（左が公理）**

結果: `A ⊢ A`

#### 最終結果: カットなし証明

```
A ⊢ A (Ax)
```

**まとめ:**

| ステップ | 適用ケース                   | カット式 | 複雑度 |
| -------- | ---------------------------- | -------- | ------ |
| 1        | Case 3.2（含意の主式カット） | A → A    | 1      |
| 2        | Case 1a（公理とのカット）    | A        | 0      |
| 3        | Case 1a（公理とのカット）    | A        | 0      |

カット式の複雑度: 1 → 0 → 0（単調減少）。3ステップで完了。

## 3. 例3: 全称量化子を含むカット除去（Case 2b → Case 3.5）

### 3.1 証明対象

```
∀x.P(x) ⊢ ∀x.P(x)
```

### 3.2 カットを含む証明

`∀x.P(x)` をカット式として、同じシーケントを2つの別々の証明から合成する:

**左前提の証明:**

```
  P(y) ⊢ P(y) (Ax)
──────────────────── (∀L: t = y)
 ∀x.P(x) ⊢ P(y)
──────────────────── (∀R: 固有変数 y — ∀x.P(x) に y は自由でない ✓)
∀x.P(x) ⊢ ∀x.P(x)
```

**右前提の証明:**

```
  P(z) ⊢ P(z) (Ax)
──────────────────── (∀L: t = z)
 ∀x.P(x) ⊢ P(z)
──────────────────── (∀R: 固有変数 z — ∀x.P(x) に z は自由でない ✓)
∀x.P(x) ⊢ ∀x.P(x)
```

**カットを含む証明図:**

```
∀x.P(x) ⊢ ∀x.P(x)        ∀x.P(x) ⊢ ∀x.P(x)
────────────────────────────────────────────────── (Cut: ∀x.P(x))
              ∀x.P(x) ⊢ ∀x.P(x)
```

- **カット式:** `∀x.P(x)`（全称量化式、複雑度 1）

### 3.3 カット除去の実行

#### ステップ1: カットの分析

- **カット式:** `∀x.P(x)`（全称量化式、複雑度 1）
- **左前提の最終規則:** `∀R`（後件に `∀x.P(x)` を導入）→ カット式が **主式**
- **右前提の最終規則:** `∀R`（後件に `∀x.P(x)` を導入）→ カット式は **前件** にあるので、∀R にとって非主式
- 右前提でカット式は非主式 → **Case 2b（非主式、右を上に持ち上げ）**

#### ステップ2: Case 2b の適用

右前提の最終規則 `∀R` をカットの外側に持ち上げる:

```
∀x.P(x) ⊢ ∀x.P(x)    ∀x.P(x) ⊢ P(z) (∀R の前提)
────────────────────────────────────────────────────── (Cut: ∀x.P(x))
                ∀x.P(x) ⊢ P(z)
             ──────────────────── (∀R: z)
              ∀x.P(x) ⊢ ∀x.P(x)
```

新しいカット: 左前提 `∀x.P(x) ⊢ ∀x.P(x)` と右前提 `∀x.P(x) ⊢ P(z)`

#### ステップ3: 新カットの分析

右前提 `∀x.P(x) ⊢ P(z)` は `∀L` で導出されている（`P(z) ⊢ P(z)` から）。

- **カット式:** `∀x.P(x)`（複雑度 1）
- **左前提の最終規則:** `∀R` → カット式が **主式**（後件に導入）
- **右前提の最終規則:** `∀L` → カット式が **主式**（前件で分解）
- 両方で主式 → **Case 3.5（全称量化の主式カット）**

#### ステップ4: Case 3.5 の適用

Case 3.5:

```
カット式: ∀x.B  (B = P(x))

左の ∀R の前提: ∀x.P(x) ⊢ P(y)    (固有変数 y)
右の ∀L の前提: P(z) ⊢ P(z)        (項 t = z)
```

変換: 固有変数 `y` を項 `t = z` で置換し、`P(z)` でカットする:

- 左の前提を `y → z` で置換: `∀x.P(x) ⊢ P(z)`
- 右の ∀L の前提: `P(z) ⊢ P(z)`

```
∀x.P(x) ⊢ P(z)    P(z) ⊢ P(z) (Ax)
───────────────────────────────────── (Cut: P(z))
          ∀x.P(x) ⊢ P(z)
```

カット式が `P(z)`（原子式、複雑度 0）に **複雑度が減少**。

左前提を展開:

```
  P(z) ⊢ P(z) (Ax)
──────────────────── (∀L: t = z)
 ∀x.P(x) ⊢ P(z)          P(z) ⊢ P(z) (Ax)
──────────────────────────────────────────── (Cut: P(z))
            ∀x.P(x) ⊢ P(z)
```

#### ステップ5: Case 2a の適用

- **カット式:** `P(z)`（原子式）
- **左前提の最終規則:** `∀L` — カット式 `P(z)` は ∀L の前提の後件から来た式 → 非主式
- **適用ケース:** **Case 2a（非主式、左を上に持ち上げ）**

∀L をカットの外側に持ち上げる:

```
P(z) ⊢ P(z) (Ax)    P(z) ⊢ P(z) (Ax)
──────────────────────────────────────── (Cut: P(z))
           P(z) ⊢ P(z)
        ──────────────── (∀L: t = z)
         ∀x.P(x) ⊢ P(z)
```

#### ステップ6: Case 1a の適用

```
P(z) ⊢ P(z) (Ax)    P(z) ⊢ P(z) (Ax)
──────────────────────────────────────── (Cut: P(z))
           P(z) ⊢ P(z)
```

- **左前提:** 公理 → **Case 1a**

結果: `P(z) ⊢ P(z)`

#### 最終結果: カットなし証明

ステップ2で外側に出した ∀R と、ステップ5で外側に出した ∀L を組み合わせる:

```
  P(z) ⊢ P(z) (Ax)
──────────────────── (∀L: t = z)
 ∀x.P(x) ⊢ P(z)
──────────────────── (∀R: 固有変数 z)
∀x.P(x) ⊢ ∀x.P(x)
```

**まとめ:**

| ステップ | 適用ケース                          | カット式       | 複雑度 |
| -------- | ----------------------------------- | -------------- | ------ |
| 1        | Case 2b（非主式、右を上に持ち上げ） | ∀x.P(x)        | 1      |
| 2        | Case 3.5（全称量化の主式カット）    | ∀x.P(x) → P(z) | 1 → 0  |
| 3        | Case 2a（非主式、左を上に持ち上げ） | P(z)           | 0      |
| 4        | Case 1a（公理とのカット）           | P(z)           | 0      |

## 4. 例4: 複数のカットが入れ子になっている場合

### 4.1 証明対象

```
A ⊢ A
```

### 4.2 カットを含む証明

`A ∧ A`（内側のカット）と `A → A`（外側のカット）の2つの補題を入れ子に経由する:

**内側のカット:**

```
                             A ∧ A ⊢ A ∧ A (Ax)
A ⊢ A (Ax)   A ⊢ A (Ax)   ──────────────────── (∧L₁)
────────────────────────── (∧R)    A ∧ A ⊢ A
      A ⊢ A ∧ A
──────────────────────────────────────────────── (Cut: A ∧ A)
                  A ⊢ A
```

**外側のカット:**

```
                                               A ⊢ A (Ax)
                                            ────────────── (WL)
                               A ⊢ A (Ax)    A, A ⊢ A
  [内側カット → A ⊢ A]       ──────────────────────────── (→L: A → A)
 ────────────── (→R)              A → A, A ⊢ A
  ⊢ A → A
──────────────────────────────────────────────── (Cut: A → A)
                    A ⊢ A
```

- **カット階数:** 1（`A → A` と `A ∧ A` はともに複雑度 1）

### 4.3 カット除去の実行

アルゴリズムはボトムアップで処理するため、**内側（葉に近い）のカットから先に処理**する。

#### フェーズ1: 内側のカット除去（Cut: A ∧ A）

##### ステップ1: カットの分析

- **カット式:** `A ∧ A`（連言、複雑度 1）
- **左前提の最終規則:** `∧R` → カット式 `A ∧ A` が主式
- **右前提の最終規則:** `∧L₁` → カット式 `A ∧ A` が主式
- 両方で主式 → **Case 3.3（連言の主式カット）**

##### ステップ2: Case 3.3 の適用

カット式 `B ∧ C`（B = A, C = A）:

- ∧R の左前提: `A ⊢ A` (B = A を後件に持つ)
- ∧R の右前提: `A ⊢ A` (C = A を後件に持つ)
- ∧L₁ の前提: `A ⊢ A` (B = A を前件に持つ)

∧L₁ は左の連言成分 B を取るので、B = A でカット:

```
A ⊢ A (Ax, ∧R の左前提)    A ⊢ A (Ax, ∧L₁ の前提)
────────────────────────────────────────────────────── (Cut: A)
                    A ⊢ A
```

カット式が `A`（原子式、複雑度 0）に減少。

##### ステップ3: Case 1a の適用

- **左前提:** `A ⊢ A` — 公理 → **Case 1a**
- 結果: `A ⊢ A`

**内側のカット除去完了。**

#### フェーズ2: 外側のカット除去（Cut: A → A）

内側のカット除去後、左前提は:

```
  A ⊢ A (Ax)
────────────── (→R)
 ⊢ A → A
```

外側のカット:

```
 ⊢ A → A                    A → A, A ⊢ A
────────────────────────────────────────── (Cut: A → A)
              A ⊢ A
```

##### ステップ4: カットの分析

- **カット式:** `A → A`（含意、複雑度 1）
- **左前提の最終規則:** `→R` → カット式 `A → A` が主式
- **右前提の最終規則:** `→L` → カット式 `A → A` が主式
- 両方で主式 → **Case 3.2（含意の主式カット）**

##### ステップ5: Case 3.2 の適用

（例2と同じパターン。B = A, C = A）

- →R の前提: `A ⊢ A`
- →L の左前提: `A ⊢ A`
- →L の右前提: `A, A ⊢ A`

2つのカットに分解:

```
                 A ⊢ A (Ax)    A, A ⊢ A (WL of Ax)
               ──────────────────────────────────── (Cut: A)
A ⊢ A (Ax)                  A ⊢ A
──────────────────────────────────── (Cut: A)
             A ⊢ A
```

##### ステップ6: 第1カット — Case 1a

左前提が公理 → 結果: `A ⊢ A`

##### ステップ7: 第2カット — Case 1a

左前提が公理 → 結果: `A ⊢ A`

#### 最終結果: カットなし証明

```
A ⊢ A (Ax)
```

**まとめ:**

| フェーズ  | ステップ | 適用ケース                   | カット式  | 複雑度 |
| --------- | -------- | ---------------------------- | --------- | ------ |
| 1（内側） | 1-2      | Case 3.3（連言の主式カット） | A ∧ A → A | 1 → 0  |
| 1（内側） | 3        | Case 1a（公理とのカット）    | A         | 0      |
| 2（外側） | 4-5      | Case 3.2（含意の主式カット） | A → A → A | 1 → 0  |
| 2（外側） | 6        | Case 1a（公理とのカット）    | A         | 0      |
| 2（外側） | 7        | Case 1a（公理とのカット）    | A         | 0      |

処理順序: 内側（葉に近い）カットから先に処理 → 外側のカット。

## 5. 証明サイズの増大について

### 5.1 上の例での観察

上の例では、カット除去による証明サイズの増大は穏やかだった:

| 例                    | カットあり証明のノード数 | カットなし証明のノード数 | 変化 |
| --------------------- | ------------------------ | ------------------------ | ---- |
| 例1（公理カット）     | 6                        | 3                        | 減少 |
| 例2（含意カット）     | 7                        | 1                        | 減少 |
| 例3（全称量化カット） | 8                        | 3                        | 減少 |
| 例4（入れ子カット）   | 12                       | 1                        | 減少 |

これらの例では、カット除去で証明がむしろ**小さく**なった。これは「不必要な遠回り」を除去した結果である。

### 5.2 証明が爆発的に大きくなるケース

しかし、カットが本質的に必要な場合（すなわち、カットによって証明が大幅に短縮されている場合）、カット除去で証明サイズが**指数的に増大**する。

**典型的なパターン: 縮約（Contraction）との組み合わせ**

カット除去の Case 4c（縮約とのカット）では:

```
Cut(A):
  左:  ... ⊢ A, A  (A が後件に2回)
  右:  A, A, ... ⊢ ...  (A が前件に2回 → 縮約で1つに)
```

この場合、1つのカットが2つのカットに分解され、それぞれの部分証明が**複製**される。これを n 回繰り返すと、証明サイズは 2^n 倍になりうる。

**理論的下界（Statman 1979）:**

カット除去によるLKの証明サイズの増大は、一般に**非初等的**（non-elementary）、すなわち:

```
サイズ増大 ≥ 2^(2^(2^...^n))   （2のタワー、高さ n に比例）
```

これは、カットが「圧縮」の役割を果たしており、繰り返し使われる補題を1度だけ証明して参照できるためである。カット除去はこの圧縮を解いて展開するため、本質的にサイズが爆発する。

### 5.3 教訓

- カット除去定理は**存在定理**として重要（カットなしの証明が存在することを保証）
- 部分式性やその帰結（一貫性証明、決定手続き等）の理論的基盤を提供
- 実用的には、カットを含む証明のほうが**遥かに小さく**効率的
- 自動証明探索では、カットなし証明の探索空間が限定されるメリットがある

> **参考:** Statman, R. "Lower bounds on Herbrand's theorem." Proceedings of the AMS, 1979.
> Orevkov, V.P. "Lower bounds for increasing complexity of derivations after cut elimination." Journal of Soviet Mathematics, 1982.
