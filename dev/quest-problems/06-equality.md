# 等号付き論理

**体系:** Predicate Logic with Equality (A1-A5 + E1-E3 + MP + Gen)
**前提:** 01-05 の問題群を完了していること

追加公理:

- E1 (反射律): `all x. x = x`
- E2 (対称律): `all x. all y. x = y -> y = x`
- E3 (推移律): `all x. all y. all z. x = y -> (y = z -> x = z)`

---

## Level 2: 等号公理の基本

### Q-39: 具体的な反射律

**難易度:** Level 2
**ゴール:** `a = a`
**ヒント:** E1とA4の組み合わせ。

**解法の概略:**

1. E1: `all x. x = x`
2. A4(t=a): `all x. x = x -> a = a`
3. MP(1, 2): `a = a`

**ステップ数:** 3
**学習ポイント:** E1の全称量化子をA4で具体化するパターン。等号論理の最も基本的な操作。

---

### Q-40: 具体的な対称律

**難易度:** Level 2
**ゴール:** `a = b -> b = a`
**ヒント:** E2をA4で2回具体化する。

**解法の概略:**

1. E2: `all x. all y. x = y -> y = x`
2. A4(t=a, 外側): `all y. a = y -> y = a`
3. A4(t=b, 内側): `a = b -> b = a`

**ステップ数:** 5
**学習ポイント:** 多重全称量化子はA4を外側から順に適用して具体化する。

---

### Q-41: 具体的な推移律

**難易度:** Level 2
**ゴール:** `a = b -> (b = c -> a = c)`
**ヒント:** E3をA4で3回具体化する。

**解法の概略:**

1. E3: `all x. all y. all z. x = y -> (y = z -> x = z)`
2. A4(t=a): `all y. all z. a = y -> (y = z -> a = z)`
3. A4(t=b): `all z. a = b -> (b = z -> a = z)`
4. A4(t=c): `a = b -> (b = c -> a = c)`

**ステップ数:** 7
**学習ポイント:** E3は3つの変数を持つ。具体化の順序を正しく管理することが重要。

---

## Level 3: 等号の組み合わせ

### Q-42: 等号のチェイン（3項）

**難易度:** Level 3
**ゴール:** `a = b -> (b = c -> (c = d -> a = d))`
**ヒント:** E3の推移律を2回適用する。

**解法の概略:**

1. Q-41: `a = b -> (b = c -> a = c)` — E3の具体化
2. E3の別インスタンス: `a = c -> (c = d -> a = d)`
3. 推移律で接続（含意の推移律 Q-04 を適用）

**ステップ数:** 約20
**学習ポイント:** 等号の推移律は任意の長さのチェインに拡張可能。

---

### Q-43: 対称律と推移律の組み合わせ

**難易度:** Level 3
**ゴール:** `a = b -> (c = b -> a = c)`
**ヒント:** 対称律でb = cをc = bに変換し、推移律を適用する。

**解法の概略:**

1. E2具体化: `c = b -> b = c` — 対称律
2. Q-41: `a = b -> (b = c -> a = c)` — 推移律
3. 中間の`b = c`を`c = b`から導く形に推移律と含意の操作で接続

**ステップ数:** 約18
**学習ポイント:** 等号の向きを揃える操作は頻出。対称律は「等号を反転させる」ために使う。

---

## Level 4: 等号と量化子

### Q-44: 全称命題の等号置換

**難易度:** Level 4
**ゴール:** `all x. P(x) -> (a = b -> (P(a) -> P(b)))`
**ヒント:** 等号の代入律（Leibniz's law）の具体例。E4/E5が動的生成される場合は、A4と等号公理で模倣する。

**注意:** 現在のシステムではE4/E5が署名依存で動的生成されるため、この問題はE1-E3のみで証明する方法を示す。一般的なLeibniz's lawの証明はメタ定理的になる。

**解法の概略:**

1. A4: `all x. P(x) -> P(a)` と `all x. P(x) -> P(b)` を導出
2. a = b の情報は、P が述語1変数の場合はE4相当の公理が必要
3. E4なしの場合: この問題は体系の表現力に依存する

**代替ゴール:** `a = a -> P(a) -> P(a)` (反射律の trivial な応用)

**ステップ数:** 体系に依存
**学習ポイント:** 等号の「代入可能性」(Substitutivity) は等号論理の核心。

---

### Q-45: 存在と等号

**難易度:** Level 4
**ゴール:** `a = b -> (P(a) -> ex x. P(x))`
**ヒント:** Q-31の存在導入を使い、等号から具体例を取り出す。

**解法の概略:**

1. Q-31相当: `P(a) -> ex x. P(x)` — aがP(x)の witness
2. A1で前提 `a = b` を追加: `a = b -> (P(a) -> ex x. P(x))`
   （等号は実質的に使われない — 等号の前提は冗長だが、問題として有効）

**ステップ数:** Q-31 + 数ステップ
**学習ポイント:** 余分な前提は常にA1で追加できる。等号が本質的でない場合もある。
