# 命題論理の中級

**体系:** Łukasiewicz (A1-A3 + MP)
**前提:** 01-propositional-basics.md の問題群を完了していること

ここでは、基礎編で証明した補題（恒等律、推移律など）を活用して、より複雑な定理を証明する。
Hilbert系での証明は「演繹定理」を手動で模倣する形で進むことが多い。

---

## Level 2: 推移律の活用

### Q-08: 推移律の3段チェイン

**難易度:** Level 2
**ゴール:** `(phi -> psi) -> ((psi -> chi) -> ((chi -> theta) -> (phi -> theta)))`
**ヒント:** 推移律 (Q-04) を2回適用する。

**解法の概略:**

1. 推移律: `(phi -> psi) -> ((psi -> chi) -> (phi -> chi))` (Q-04)
2. 推移律: `(phi -> chi) -> ((chi -> theta) -> (phi -> theta))` (Q-04のインスタンス)
3. これらを組み合わせて3段チェインを構成

**ステップ数:** 約25
**学習ポイント:** 推移律は任意の長さのチェインに拡張できる。

---

### Q-09: 自己適用 (W Combinator)

**難易度:** Level 2
**ゴール:** `(phi -> (phi -> psi)) -> (phi -> psi)`
**ヒント:** Q-06と同じ。S公理で$\varphi$の2重要求を1回に畳む。

**解法の概略:** Q-06と同一。

**ステップ数:** 約12
**学習ポイント:** W combinator: `Wxy = xyy`。引数を複製する操作に対応。

---

## Level 3: 複合的な証明

### Q-10: B combinator (合成)

**難易度:** Level 3
**ゴール:** `(psi -> chi) -> ((phi -> psi) -> (phi -> chi))`
**ヒント:** 推移律 (Q-04) の前提を入れ替えたもの。C combinator (Q-07) と推移律の組合せ。

**解法の概略:**

1. 推移律: `(phi -> psi) -> ((psi -> chi) -> (phi -> chi))`
2. 含意の交換 (Q-07) を推移律に適用して前提を入れ替える

**ステップ数:** 約20
**学習ポイント:** B combinator: `Bxyz = x(yz)`。関数合成に対応。

---

### Q-11: 前提の合流

**難易度:** Level 3
**ゴール:** `(phi -> (psi -> chi)) -> ((phi -> psi) -> (phi -> chi))`
**ヒント:** これはA2そのもの。公理のインスタンスとして即座に得られる。

**解法の概略:**

1. A2をそのまま使う

**ステップ数:** 1 (公理インスタンス)
**学習ポイント:** 一見難しそうに見えても、公理のインスタンスであることがある。メタ変数への代入パターンを見抜く力が重要。

---

### Q-12: 含意の左結合化

**難易度:** Level 3
**ゴール:** `((phi -> psi) -> (phi -> chi)) -> (phi -> (psi -> chi))`
**ヒント:** A1で`psi`を`phi -> ...`の中に持ち上げ、S公理とQ-07で整理する。

**解法の概略:**

1. A1: `psi -> (phi -> psi)` — $\psi$を$\varphi$の前提に持ち上げ
2. 推移律で `(phi -> psi) -> (phi -> chi)` の前提側を変形
3. 含意の交換 (Q-07) で整理

**ステップ数:** 約20
**学習ポイント:** 含意の「右結合」と「左結合」を相互変換する技法。演繹定理の逆方向。

---

### Q-13: Frege の定理

**難易度:** Level 3
**ゴール:** `(phi -> (psi -> chi)) -> ((phi -> psi) -> (phi -> chi))`
**ヒント:** A2と同一だが、ここでは「Fregeの定理」として名前を付ける。

**解法の概略:** A2のインスタンス。

**ステップ数:** 1
**学習ポイント:** Frege は歴史上最初にこの法則を明示的に公理として採用した。

---

### Q-14: 二重含意の分配

**難易度:** Level 3
**ゴール:** `(phi -> psi) -> ((phi -> (psi -> chi)) -> (phi -> chi))`
**ヒント:** A2の結論部分に推移律を適用する形。

**解法の概略:**

1. A2: `(phi -> (psi -> chi)) -> ((phi -> psi) -> (phi -> chi))`
2. 含意の交換 (Q-07) で前提の順序を入れ替え

**ステップ数:** 約18
**学習ポイント:** 前提の並び替えはHilbert系では頻出操作。慣れが必要。
