## Level 4: 量化子の相互作用

### Q-33: 全称量化子と含意の交換

**難易度:** Level 4
**ゴール:** `(all x. (P(x) -> Q(x))) -> ((all x. P(x)) -> (all x. Q(x)))`
**ヒント:** Q-26/Q-29 と同一の定理。復習として再度証明する。

**解法の概略:** Q-26と同一。

**ステップ数:** 約15
**学習ポイント:** この形は述語論理で最も頻出するパターン。

### Q-34: 存在量化子の否定

**難易度:** Level 4
**ゴール:** `~(ex x. P(x)) -> all x. ~P(x)`
**ヒント:** $\exists$x.P(x) $\equiv \lnot \forall x. \lnot$P(x) の定義を使い、二重否定除去と組み合わせる。

**解法の概略:**
定義展開: `~(ex x. P(x))` = `~~(all x. ~P(x))`

1. 二重否定除去 (Q-17): `~~(all x. ~P(x)) -> all x. ~P(x)`
2. これが証明。

**ステップ数:** Q-17のステップ数（公理のインスタンスを適切に選ぶ）
**学習ポイント:** 存在量化子の定義 `$\exists x. \varphi \equiv \lnot \forall x. \lnot \varphi$` を理解していれば、多くの存在量化子の性質は全称量化子に帰着できる。

### Q-35: 全称量化子の否定

**難易度:** Level 4
**ゴール:** `~(all x. P(x)) -> ex x. ~P(x)`
**ヒント:** ゴールの定義を展開すると `~(all x. P(x)) -> ~(all x. ~~P(x))`。Modus Tollens と二重否定導入を使う。

**解法の概略:**

1. 二重否定導入 `P(x) -> ~~P(x)` をGenで汎化
2. Q-26で `all x. P(x) -> all x. ~~P(x)` を導出
3. Modus Tollens: `~(all x. ~~P(x)) -> ~(all x. P(x))`
4. 対偶の逆 (A3): `~(all x. P(x)) -> ~(all x. ~~P(x))`
   ん？方向が逆。もう少し工夫が必要。
5. 実際には Modus Tollens を `all x. P(x) -> all x. ~~P(x)` に適用:
   `~(all x. ~~P(x)) -> ~(all x. P(x))`
6. A3 で逆を取る: `~(all x. P(x)) -> ~~(all x. ~~P(x))` ...ではなく
   `~(all x. P(x)) -> ~(all x. ~~P(x))` を直接は出しにくい

   正しいアプローチ: 対偶を使わず、爆発律的な構成が必要。完全な証明は解答ノートで提供。

**ステップ数:** 約30
**学習ポイント:** 全称量化子の否定 $\to$ 存在量化子は、Hilbert系では意外と証明が難しい。

## Level 5: 複合的な述語論理の定理

### Q-36: 存在量化子の含意分配

**難易度:** Level 5
**ゴール:** `all x. (P(x) -> Q(x)) -> (ex x. P(x) -> ex x. Q(x))`
**ヒント:** 存在の定義を展開し、全称の含意分配と否定の操作を組み合わせる。

**解法の概略:**
定義展開:

- `ex x. P(x)` = `~(all x. ~P(x))`
- `ex x. Q(x)` = `~(all x. ~Q(x))`

1. `all x. (P(x) -> Q(x))` から Modus Tollens で `~Q(x) -> ~P(x)` を導出
2. Genで汎化、A5で量化子を出す
3. `all x. ~Q(x) -> all x. ~P(x)` を導出
4. Modus Tollens: `~(all x. ~P(x)) -> ~(all x. ~Q(x))`
5. 推移律で全体を接続

**ステップ数:** 40以上
**学習ポイント:** 存在量化子の扱いは本質的に否定＋全称の操作。

### Q-37: 量化子の入れ替え（存在$\to$全称方向）

**難易度:** Level 5
**ゴール:** `ex x. all y. P(x, y) -> all y. ex x. P(x, y)`
**ヒント:** 存在の定義展開と全称の順序交換を組み合わせる。

**解法の概略:**
複雑な定義展開と多段階の証明が必要。完全な証明は解答ノートで提供。

**注意:** 逆方向 `all y. ex x. P(x, y) -> ex x. all y. P(x, y)` は一般に成り立たない。

**ステップ数:** 50以上
**学習ポイント:** $\exists x. \forall y \to \forall y. \exists$x は成り立つが、逆は成り立たない。これは述語論理の重要な非対称性。

### Q-38: Drinker's Paradox

**難易度:** Level 5
**ゴール:** `ex x. (P(x) -> all y. P(y))`
**ヒント:** 古典論理でのみ成り立つ有名な定理。「酒場に必ず一人、その人が飲めば全員飲む人がいる」。

**解法の概略:**
排中律 `$\forall$y.P(y) $\lor \lnot \forall$y.P(y)` から場合分け:

1. `$\forall$y.P(y)` の場合: 任意の a について `P(a) $\to \forall$y.P(y)` が成り立つ（A1で前提を無視）
2. `$\lnot \forall$y.P(y)` の場合: `$\lnot$P(a)` となる a が存在し、`P(a) $\to \forall$y.P(y)` は前提が偽なので爆発律で成り立つ

ただし Hilbert 系での形式化は非常に複雑。

**ステップ数:** 60以上
**学習ポイント:** 古典論理の「非構成的」な性質の典型例。存在を証明するが、具体的な witness を構成しない。
