# 等号付き論理 — 解答ノート

**体系:** Predicate Logic with Equality (A1-A5 + E1-E3 + MP + Gen)

**等号公理:**

- E1 (反射律): `all x. x = x`
- E2 (対称律): `all x. all y. x = y -> y = x`
- E3 (推移律): `all x. all y. all z. x = y -> (y = z -> x = z)`

**既証の補題:**

- 命題論理: Id, HS, W, C, B, DNI, DNE, MT, EFQ
- 述語論理: Dist$\forall , \forall$swap, $\exists$intro

## Q-42: 等号のチェイン（3項）

**ゴール:** `a = b -> (b = c -> (c = d -> a = d))`

### 証明

| #   | 式                          | 根拠                                                |
| --- | --------------------------- | --------------------------------------------------- |
| 1   | `a = b -> (b = c -> a = c)` | Q-41                                                |
| 2   | `a = c -> (c = d -> a = d)` | Q-41 [a := a, b := c, c := d] — E3 の別インスタンス |

`a = b -> (b = c -> (c = d -> a = d))` を構成するには:

- `a = b` と `b = c` から `a = c` (Q-41)
- `a = c` と `c = d` から `a = d` (Q-41 の別インスタンス)

形式的に:

| 3 | `(b = c -> a = c) -> ((a = c -> (c = d -> a = d)) -> (b = c -> (c = d -> a = d)))` | HS |
| 4 | `(a = c -> (c = d -> a = d)) -> (b = c -> (c = d -> a = d))` | — `a = b` を仮定した下で MP(1に a=b, 3) |

演繹定理的:

- `a = b` を仮定 $\to$ Q-41 から `b = c -> a = c`
- HS(上記, Q-41のインスタンス2): `b = c -> (c = d -> a = d)`

形式化:

| #   | 式                                                                                 | 根拠                                                          |
| --- | ---------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| 1   | `a = b -> (b = c -> a = c)`                                                        | Q-41                                                          |
| 2   | `a = c -> (c = d -> a = d)`                                                        | E3 のインスタンス (Q-41 と同様に導出)                         |
| 3   | `(b = c -> a = c) -> ((a = c -> (c = d -> a = d)) -> (b = c -> (c = d -> a = d)))` | HS [alpha := b = c, beta := a = c, gamma := (c = d -> a = d)] |
| 4   | `(a = c -> (c = d -> a = d)) -> ((b = c -> a = c) -> (b = c -> (c = d -> a = d)))` | C で ステップ3 の前提を入れ替え                               |
| 5   | `(b = c -> a = c) -> (b = c -> (c = d -> a = d))`                                  | MP(2, 4)                                                      |
| 6   | `a = b -> (b = c -> (c = d -> a = d))`                                             | HS(1, 5)                                                      |

ステップ3-5の代わりに、B を使って簡潔に:

| 3 | `(a = c -> (c = d -> a = d)) -> ((b = c -> a = c) -> (b = c -> (c = d -> a = d)))` | B [alpha := b = c, beta := a = c, gamma := c = d -> a = d] |
| 4 | `(b = c -> a = c) -> (b = c -> (c = d -> a = d))` | MP(2, 3) |
| 5 | `a = b -> (b = c -> (c = d -> a = d))` | HS(1, 4) |

**ステップ数:** 5 + Q-41 x 2 + HS + B = 約20

## Q-43: 対称律と推移律の組み合わせ

**ゴール:** `a = b -> (c = b -> a = c)`

### 証明

| #   | 式                          | 根拠                                      |
| --- | --------------------------- | ----------------------------------------- |
| 1   | `c = b -> b = c`            | Q-40 [a := c, b := b] — E2 のインスタンス |
| 2   | `a = b -> (b = c -> a = c)` | Q-41                                      |

`a = b -> (c = b -> a = c)` を構成:

- `c = b` から `b = c` (ステップ1)
- `a = b` と `b = c` から `a = c` (ステップ2)

形式化:

| 3 | `(c = b -> b = c) -> ((b = c -> ...) -> (c = b -> ...))` | B |

B [alpha := c = b, beta := b = c, gamma := a = c]:
`(b = c -> a = c) -> ((c = b -> b = c) -> (c = b -> a = c))`

C で前提を入れ替えて:
`(c = b -> b = c) -> ((b = c -> a = c) -> (c = b -> a = c))`

| 3 | `(b = c -> a = c) -> ((c = b -> b = c) -> (c = b -> a = c))` | B [alpha := c = b, beta := b = c, gamma := a = c] |
| 4 | `(c = b -> b = c) -> ((b = c -> a = c) -> (c = b -> a = c))` | C で 3 の前提を入れ替え |
| 5 | `(b = c -> a = c) -> (c = b -> a = c)` | MP(1, 4) |
| 6 | `a = b -> (c = b -> a = c)` | HS(2, 5) — ステップ2 の結論 `b = c -> a = c` をステップ5 に通す |

Wait: ステップ2 は `a = b -> (b = c -> a = c)`。これは `a = b` の下で `b = c -> a = c` を与える。
ステップ5 は `(b = c -> a = c) -> (c = b -> a = c)`。
HS: `a = b -> (c = b -> a = c)`。

でも HS は `(A -> B) -> ((B -> C) -> (A -> C))` で、ここでは:
A = a = b, B = b = c -> a = c, C = c = b -> a = c ではない。
ステップ2: `a = b -> (b = c -> a = c)`
ステップ5: `(b = c -> a = c) -> (c = b -> a = c)`

B: `(B -> C) -> ((A -> B) -> (A -> C))`
B = b = c -> a = c, C = c = b -> a = c, A = a = b:
`((b = c -> a = c) -> (c = b -> a = c)) -> ((a = b -> (b = c -> a = c)) -> (a = b -> (c = b -> a = c)))`

| 6 | `((b = c -> a = c) -> (c = b -> a = c)) -> ((a = b -> (b = c -> a = c)) -> (a = b -> (c = b -> a = c)))` | B |
| 7 | `(a = b -> (b = c -> a = c)) -> (a = b -> (c = b -> a = c))` | MP(5, 6) |
| 8 | `a = b -> (c = b -> a = c)` | MP(2, 7) |

**ステップ数:** 8 + 補題展開 = 約18

## Q-44: 全称命題の等号置換

**ゴール:** `all x. P(x) -> (a = b -> (P(a) -> P(b)))`

### 証明

**注意:** E4/E5 (関数/述語の合同性公理) が体系に含まれていれば直接使えるが、ここでは E1-E3 のみで証明する。

`all x. P(x)` を仮定すれば `P(a)` と `P(b)` の両方が A4 で得られるので、`a = b` は実質使わない。

| #   | 式                                            | 根拠                                   |
| --- | --------------------------------------------- | -------------------------------------- |
| 1   | `all x. P(x) -> P(b)`                         | A4 [t := b]                            |
| 2   | `P(b) -> (P(a) -> P(b))`                      | A1 [phi := P(b), psi := P(a)]          |
| 3   | `all x. P(x) -> (P(a) -> P(b))`               | HS(1, 2)                               |
| 4   | `(P(a) -> P(b)) -> (a = b -> (P(a) -> P(b)))` | A1 [phi := P(a) -> P(b), psi := a = b] |
| 5   | `all x. P(x) -> (a = b -> (P(a) -> P(b)))`    | HS(3, 4)                               |

**ステップ数:** 5 + HS = 約16

**注:** この問題では `a = b` の前提は冗長。等号が本質的に使われるのは E4/E5 を持つ体系でのみ。

## Q-45: 存在と等号

**ゴール:** `a = b -> (P(a) -> ex x. P(x))`

### 証明

`P(a) -> ex x. P(x)` は Q-31 で既証。`a = b` は冗長な前提として A1 で追加する。

| #   | 式                                                        | 根拠                                         |
| --- | --------------------------------------------------------- | -------------------------------------------- |
| 1   | `P(a) -> ex x. P(x)`                                      | Q-31 ($\exists$intro)                                |
| 2   | `(P(a) -> ex x. P(x)) -> (a = b -> (P(a) -> ex x. P(x)))` | A1 [phi := P(a) -> ex x. P(x), psi := a = b] |
| 3   | `a = b -> (P(a) -> ex x. P(x))`                           | MP(1, 2)                                     |

**ステップ数:** 3 + Q-31 = 約28

**注:** Q-44 同様、`a = b` の前提は E4/E5 なしでは冗長。等号の代入可能性 (Leibniz's law) を本質的に使うには E4/E5 公理が必要。
