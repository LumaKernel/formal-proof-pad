# 述語論理の基礎

**体系:** Predicate Logic (A1-A5 + MP + Gen)
**前提:** 01-03 の命題論理問題群を完了していること

追加公理:

- A4 (全称例化): ∀x.φ → φ[t/x] （tがxに対してφ内で自由）
- A5 (全称分配): ∀x.(φ → ψ) → (φ → ∀x.ψ) （x ∉ FV(φ)）

追加推論規則:

- Gen (汎化): φ から ∀x.φ を導出

---

## Level 2: 全称量化子の基本

### Q-25: 全称例化の直接適用

**難易度:** Level 2
**ゴール:** `all x. P(x) -> P(a)`
**ヒント:** A4の直接のインスタンス。∀x.P(x) に t=a を代入。

**解法の概略:**

1. A4: `all x. P(x) -> P(a)` — 直接のインスタンス（φ=P(x), t=a）

**ステップ数:** 1 (公理インスタンス)
**学習ポイント:** A4は「全称命題から具体例を取り出す」操作。最も基本的な述語論理の推論。

---

### Q-26: 全称命題の推移律

**難易度:** Level 2
**ゴール:** `all x. (P(x) -> Q(x)) -> (all x. P(x) -> all x. Q(x))`
**ヒント:** A5で全称量化子を分配し、A4で具体化してMPを適用し、Genで再汎化する。

**解法の概略:**

1. A4: `all x. (P(x) -> Q(x)) -> (P(x) -> Q(x))` — 全称の中身を取り出す
2. A4: `all x. P(x) -> P(x)` — 同様
3. 推移律で接続してMP: P(x) → Q(x) と P(x) から Q(x)
4. Gen: Q(x) から all x. Q(x)
5. A5を使って前提を全称の外に出す

**ステップ数:** 約15
**学習ポイント:** 全称量化子は「含意を保存する」。この性質は述語論理の基本。

---

### Q-27: 汎化と含意の組み合わせ

**難易度:** Level 2
**ゴール:** `(all x. P(x)) -> (all x. (Q(x) -> P(x)))`
**ヒント:** A4でP(x)を取り出し、A1で前提Q(x)を追加し、Genで汎化する。

**解法の概略:**

1. A4: `all x. P(x) -> P(x)` — 具体化
2. A1: `P(x) -> (Q(x) -> P(x))` — 前提を追加
3. 推移律: `all x. P(x) -> (Q(x) -> P(x))`
4. A5: `all x. (all x. P(x) -> (Q(x) -> P(x)))` を使って...

   実際は A5 の適用条件「x ∉ FV(φ)」に注意が必要。
   `all x. P(x)` の中で x は束縛されているので、FV(all x. P(x)) には x は含まれない。
   よって A5 が適用可能。

**ステップ数:** 約12
**学習ポイント:** A5の適用条件 `x ∉ FV(φ)` の理解が重要。束縛変数と自由変数の区別。

---

## Level 3: A5の活用

### Q-28: 全称量化子の入れ子

**難易度:** Level 3
**ゴール:** `all x. all y. P(x, y) -> all y. all x. P(x, y)`
**ヒント:** A4で外側の全称を外し、内側の全称を外し、Genで逆順に汎化する。

**解法の概略:**

1. A4(外側): `all x. all y. P(x, y) -> all y. P(x, y)`
2. A4(内側): `all y. P(x, y) -> P(x, y)`
3. 推移律: `all x. all y. P(x, y) -> P(x, y)`
4. Gen(x): `all x. all y. P(x, y) -> all x. P(x, y)` — A5で量化子を前提の外に
5. Gen(y) + A5: `all x. all y. P(x, y) -> all y. all x. P(x, y)`

**ステップ数:** 約18
**学習ポイント:** 全称量化子の順序は交換可能。これは古典・直観主義の両方で成り立つ。

---

### Q-29: 全称の含意分配

**難易度:** Level 3
**ゴール:** `all x. (P(x) -> Q(x)) -> (all x. P(x) -> all x. Q(x))`
**ヒント:** Q-26と同一だが、別の証明経路を試す。

**解法の概略:**
A5を活用したアプローチ:

1. A4: `all x. (P(x) -> Q(x)) -> (P(x) -> Q(x))`
2. A4: `all x. P(x) -> P(x)`
3. S公理の述語論理版で組み合わせ
4. Gen + A5で量化

**ステップ数:** 約15
**学習ポイント:** 同じ定理でも複数の証明経路が存在する。

---

### Q-30: 空虚な全称量化

**難易度:** Level 3
**ゴール:** `phi -> all x. phi`
**ヒント:** φにxが自由出現しない場合、Genが直接適用可能。ただしφはメタ変数なのでxが自由出現するかは未定。この問題ではφにxが自由出現しないと仮定する（A5の条件を満たす）。

**注意:** この問題は`phi`がxを含まない式でのみ有効。具体的なインスタンスとして `P(a) -> all x. P(a)` を出題する。

**具体ゴール:** `P(a) -> all x. P(a)`

**解法の概略:**

1. A1: `P(a) -> (P(a) -> P(a))` — ダミー（実際には恒等律の準備）
2. 恒等律: `P(a) -> P(a)`
3. Gen: `all x. (P(a) -> P(a))` — xはP(a)に自由出現しないのでGen可能
4. A5: `all x. (P(a) -> P(a)) -> (P(a) -> all x. P(a))` — x ∉ FV(P(a))
5. MP(3, 4): `P(a) -> all x. P(a)`

**ステップ数:** 約10
**学習ポイント:** 自由変数を含まない式はいつでも全称量化できる（空虚な量化）。

---

## Level 4: 存在量化子

### Q-31: 存在量化子の導入

**難易度:** Level 4
**ゴール:** `P(a) -> ex x. P(x)`
**ヒント:** 存在量化子は `∃x.φ ≡ ¬∀x.¬φ` として定義される。A4の対偶を取る形で証明する。

**注意:** 現在のシステムで`∃x.φ`がこの定義で処理されているか確認が必要。
パーサーは`ex x. P(x)`を直接`Existential`ノードとして解析する。

**解法の概略:**

1. A4: `all x. ~P(x) -> ~P(a)` — A4のφ=¬P(x)、t=a
2. 対偶 (Q-16): `(all x. ~P(x) -> ~P(a)) -> (~~P(a) -> ~(all x. ~P(x)))`
   — Modus Tollens を A4 に適用
3. MP: `~~P(a) -> ~(all x. ~P(x))`
4. 二重否定導入: `P(a) -> ~~P(a)`
5. 推移律: `P(a) -> ~(all x. ~P(x))`
6. `~(all x. ~P(x))` = `ex x. P(x)` (定義)

**ステップ数:** 約25
**学習ポイント:** 存在量化子は全称量化子の否定で定義できる。「具体例があれば存在する」は直感的だが、形式証明は長い。

---

### Q-32: 全称から存在への推論

**難易度:** Level 4
**ゴール:** `all x. P(x) -> ex x. P(x)`
**ヒント:** 全称を具体化し、Q-31を適用する。

**解法の概略:**

1. A4: `all x. P(x) -> P(a)` — 具体化
2. Q-31: `P(a) -> ex x. P(x)` — 存在の導入
3. 推移律: `all x. P(x) -> ex x. P(x)`

**ステップ数:** Q-31 + 数ステップ
**学習ポイント:** 「すべてに当てはまるなら、少なくとも一つに当てはまる」は当然だが、空の領域では破綻する点に注意。
