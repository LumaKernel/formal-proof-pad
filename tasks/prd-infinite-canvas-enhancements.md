# PRD: InfiniteCanvas 機能拡張計画

## 概要

Figma, Miro, tldraw, Excalidraw 等の既存 infinite canvas アプリケーションから学び、証明ワークスペースに有用な機能を段階的に追加する。

### 調査元

- Figma: 業界標準のデザインツール。ズーム/パン/選択/整列/スナップ・グリッド・キーボードショートカット
- Miro: ホワイトボード。マルチユーザー、ナビゲーション
- tldraw: OSS Canvas。パフォーマンス最適化（viewport culling, signal-based rendering）、スナップ・ガイド
- Excalidraw: OSS Canvas。スナップ、エクスポート、elbow routing

### 現在実装済みの機能

- パン（ドラッグ、タッチ）
- ズーム（ホイール、ピンチ、プリセット、フィット、リセット、UIコントロール）
- ドット/メジャーグリッド背景
- ドラッグ可能アイテム（CanvasItem）
- ベジェ曲線接続線（Connection, PortConnection）、障害物回避
- コンテキストメニュー（右クリック、ロングプレス）
- ノードメニュー、ラインメニュー
- ミニマップ（ビューポートインジケータ、クリックナビゲーション）
- コネクタポート
- 接続ラベル
- ZoomControls UI（ズームイン/アウト、プリセット、リセット、フィット）

---

## Tier 1: 必須（高インパクト、コアUX）

### 1.1 マルチセレクション（矩形選択）

- [x] 空のキャンバス領域でドラッグすると選択矩形（マーキー）が表示される
- [x] 矩形内のアイテムがすべて選択される
- [x] Shift+クリックで選択に追加/除外
- [x] 選択中アイテムのハイライト表示（アウトライン等）
- [x] 選択中アイテムの一括ドラッグ移動
- [x] Escape で選択解除
- [x] Ctrl/Cmd+A で全選択

### 1.2 キーボードショートカット

- [x] Delete/Backspace で選択アイテム削除
- [x] 矢印キーでビューポートパン（Shift+矢印で大きなステップ）
- [x] Ctrl/Cmd +/- でズームイン/アウト
- [x] Space+ドラッグでパンモード（現在のツールに関係なく）

### 1.3 証明ツリー自動レイアウト

- [x] d3-hierarchy または elkjs を使った自動ツリーレイアウト
- [x] レイアウト方向の選択（上→下 / 下→上）
- [-] ノード追加/削除時のインクリメンタルな再レイアウト
- [x] 「自動整列」ボタンで手動トリガー

### 1.4 サブツリー選択

- [x] ノード右クリック → 「サブツリーを選択」でそのノードと全子孫を選択
- [x] 選択したサブツリーの一括移動

## Tier 2: 重要（UX大幅改善）

### 2.1 スナップ・整列

- [ ] グリッドスナップ（ドラッグ時にグリッド位置に吸着）
- [ ] オブジェクトスナップ（他アイテムのエッジ/中心に整列ガイド表示）
- [ ] 選択アイテムの整列操作（左/中央/右/上/中/下）
- [ ] 等間隔分配

### 2.2 ビュー操作の拡充

- [ ] 選択範囲にズーム（Shift+2）
- [ ] ノード検索＆ナビゲーション（フォーミュラテキストで検索→該当ノードにフォーカス）
- [ ] ドラッグ中のエッジスクロール（ビューポート端に近づくと自動パン）

### 2.3 接続プレビュー・バリデーション

- [ ] ポートからドラッグ中にゴースト接続線を表示
- [ ] ホバー中のポートにスナップ
- [ ] 接続が有効/無効かをリアルタイムで視覚的にフィードバック

---

## Tier 3: 価値あり（効率・体験向上）

### 3.1 コマンドパレット

- [ ] `/` キーでコマンドパレットを開く
- [ ] 公理/推論規則をテキスト検索で素早く挿入
- [ ] キーボードドリブンなワークフロー

### 3.2 ノード複製・コピーペースト

- [ ] Ctrl/Cmd+D で選択アイテムを複製（オフセット付き）
- [ ] Ctrl/Cmd+C/V/X でクリップボード操作

### 3.3 Level-of-Detail レンダリング

- [ ] 低ズームレベルではノード内容を簡略表示（色付きブロックのみ等）
- [ ] axiom/derived/goal のステータスを色で区別

### 3.4 Viewport Culling

- [ ] 画面外のアイテムを非表示にしてパフォーマンス改善
- [ ] バッファゾーン付きでスムーズなスクロール体験

### 3.5 エクスポート

- [ ] PNG/SVG 画像エクスポート
- [ ] JSON データエクスポート（証明の共有・バックアップ用）

## Tier 4: 将来検討

- グルーピング/フレーム（サブ証明のグループ化）
- ビューポート履歴（戻る/進む）
- ビューポートブックマーク
- スマート整列ガイド（等間隔ガイド）
- 高速ドラッグ時のスナップ抑制

---

## 実装方針

- 既存の3層分離パターンを踏襲: 純粋ロジック(.ts) → React hook → UIコンポーネント(.tsx)
- 各機能は独立したモジュールとして実装し、InfiniteCanvas に合成可能にする
- Tier 1 → Tier 2 → Tier 3 の順で段階的に実装
- 各機能に対してユニットテスト + Storybook ストーリーを作成
