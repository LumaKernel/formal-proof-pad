# PRD: 自作クエスト機能

> **優先度: 低** — 他の主要タスク完了後に着手する。

## はじめに

ユーザーが独自のクエスト（証明課題）を作成・管理・共有できる機能を提供する。ビルトインクエストとは別枠で管理し、クエストの作成・編集・複製・削除・共有を可能にする。

## サブタスク

### クエスト作成

- [x] 新規クエスト作成UI
  - ゴールとなる論理式の指定（複数可）
  - 各ゴールで使ってよい公理の指定
  - 体系（論理系）の選択
  - タイトル・説明の入力
- [x] 作成したクエストからクエストノートを開始する

### ビルトインからのカスタム

- [x] ビルトインクエストを「複製して自作する」コマンド
  - ビルトインクエストを自作クエストとしてコピーし、自由に編集可能にする
  - 元のビルトインクエストとの関連は参考情報として保持（必須ではない）

### クエスト共有

- [x] クエストをJSON形式でエクスポート
- [x] JSON形式のクエストをインポート（自作クエストに追加）
- [ ] URL形式でのクエスト共有（URLにクエスト定義をエンコード）
- [ ] 共有されたクエストを受け取った場合の選択肢:
  - いきなりクエストノートとして開始する
  - 自作クエストにまず追加してから開始する
