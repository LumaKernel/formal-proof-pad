# PRD: UIスタイル改善 — 紙のような質感の実現

## 概要

現在の UI はモダンでミニマルだが、「紙のような質感」を取り入れることで、数学のノートやテキストブックを操作している感覚を強め、論理学の形式証明というテーマに合った体験を提供する。

## 設計方針

- **紙のような質感**: クリーム色/暖色系の背景、微細なテクスチャ、手書き的な線、陰影で物理的な紙の感覚を再現
- **使い勝手は変えない**: 機能的な変更はなし。見た目のみの改善
- **コンポーネントAPI の明確化**: インラインスタイルの散在をCSS変数・CSS Modulesへ整理し、責務範囲を明確にする
- **ライト/ダークモード両対応**: CSS変数ベースでテーマ切り替えを維持
- **テスト**: Storybookのスクリーンショットで毎回確認。computedな位置関係の検証

## 現状分析

### スタイリングの課題

1. **インラインスタイルの散在**: ProofWorkspace.tsx, EditableProofNode.tsx, AxiomPalette.tsx 等に大量のCSSPropertiesオブジェクトがハードコードされている
2. **ノード色がハードコード**: proofNodeUI.ts で `#5b8bd9` 等のカラーコードが直書き。テーマ非対応
3. **テクスチャなし**: フラットな背景色のみ。紙・手書きの雰囲気がない
4. **統一的なデザイントークンの不足**: コンポーネント間で微妙にスタイルが異なる

### 影響範囲のファイル

- `src/app/globals.css` — テーマCSS変数
- `src/lib/proof-pad/proofNodeUI.ts` — ノード色定義
- `src/lib/proof-pad/ProofWorkspace.tsx` — ツールバー・バナー
- `src/lib/proof-pad/EditableProofNode.tsx` — 証明ノード
- `src/lib/proof-pad/AxiomPalette.tsx` — 公理パレット
- `src/lib/infinite-canvas/InfiniteCanvas.tsx` — キャンバス背景
- `src/lib/notebook/NotebookListComponent.tsx` — ノートブック一覧
- `src/lib/quest/QuestCatalogComponent.tsx` — クエストカタログ
- `src/components/ThemeToggle/ThemeToggle.module.css` — テーマトグル

## 実装方針

- **Phase 1 を先に**: デザイントークン整理 → テクスチャ → ノード → ツールバーの順で段階的に
- **リファクタリング先行**: 色のCSS変数化は見た目を変えずに先にやる（US-S01）
- **各ステップでスクリーンショット比較**: 変更前後のスクリーンショットを `.screenshots/` に保存
- **テスト維持**: 全ての既存テスト・play関数が通ること
- **パフォーマンス注意**: テクスチャはSVGパターンまたはCSS gradientで実現。画像ファイル不使用
